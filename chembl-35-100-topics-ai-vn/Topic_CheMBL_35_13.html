```python from rdkit import Chem from rdkit.Chem import AllChem from rdkit.Chem import Descriptors

V√≠ d·ª• 3: T√≠nh to√°n v√† th√™m s·ªë li√™n k·∫øt hydro ch·∫•p nh·∫≠n (HBA) v√† cho (HBD) (Python)

```python from rdkit.Chem import Lipinski

def calculate_hba(smiles): mol = Chem.MolFromSmiles(smiles) if mol is not None: hba = Lipinski.NumHAcceptors(mol) return hba else: return None

def calculate_hbd(smiles): mol = Chem.MolFromSmiles(smiles) if mol is not None: hbd = Lipinski.NumHDonors(mol) return hbd else: return None

data['hba'] = data['canonical_smiles'].apply(calculate_hba) data['hbd'] = data['canonical_smiles'].apply(calculate_hbd)

print(data[['canonical_smiles', 'hba', 'hbd']].head()) ```
