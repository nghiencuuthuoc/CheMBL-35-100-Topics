3. M√£ ngu·ªìn SQL v√† Python m·∫´u (Sample SQL and Python Code):

4. V√≠ d·ª• code SQL v√† Python m·∫´u (Sample SQL and Python Code Examples):

V√≠ d·ª• 1: SQL - L·ªçc c√°c h·ª£p ch·∫•t c√≥ ho·∫°t t√≠nh IC50 d∆∞·ªõi 100 nM:

V√≠ d·ª• 1: Python - T√≠nh to√°n logP v√† MW s·ª≠ d·ª•ng RDKit:

```python from rdkit.Chem import Descriptors

def calculate_logp_mw(smiles): mol = Chem.MolFromSmiles(smiles) if mol is not None: logp = Descriptors.MolLogP(mol) mw = Descriptors.MolWt(mol) return logp, mw else: return None, None

df['logp'], df['mw'] = zip(*df['canonical_smiles'].apply(calculate_logp_mw)) df = df.dropna(subset=['logp', 'mw']) print(df[['canonical_smiles', 'logp', 'mw']].head()) ```

V√≠ d·ª• 2: SQL - T√¨m c√°c h·ª£p ch·∫•t t√°c ƒë·ªông l√™n m·ªôt target c·ª• th·ªÉ (v√≠ d·ª•: kinase):

V√≠ d·ª• 2: Python - Chu·∫©n h√≥a gi√° tr·ªã IC50 (pIC50):

```python def calculate_pic50(ic50): # IC50 ph·∫£i ·ªü ƒë∆°n v·ªã nM if ic50 is not None: pic50 = -np.log10(ic50 / 1e9) # Chuy·ªÉn nM v·ªÅ M return pic50 else: return None

V√≠ d·ª• 3: SQL - L·∫•y d·ªØ li·ªáu v·ªÅ c√°c h·ª£p ch·∫•t c√≥ ho·∫°t t√≠nh tr√™n m·ªôt protein c·ª• th·ªÉ (s·ª≠ d·ª•ng accession number):
